# E2E 测试自然语言使用指南

## 概述

本指南说明如何通过自然语言指令与 AI 助手交互来执行 E2E 测试。您无需记忆任何命令行，只需用日常语言描述您的需求即可。

---

## 一、基本交互模式

### 指令结构

```
我想 [执行什么操作] + [关于什么内容] + [有什么特殊要求]
```

### 示例

- "帮我运行所有测试"
- "我想测试登录功能"
- "请检查系统管理模块，用无头模式运行"

---

```

- 可能原因：前端代码变更
- 位置：src/views/user/login.vue
- 需要我帮您检查代码吗？
```

## 十、注意事项

1. **随时打断**：测试过程中您可以说"停止"来中断执行
2. **随时提问**：测试执行期间可以询问进度或详情
3. **自然表达**：不必拘泥于固定格式，用您觉得舒服的方式表达即可
4. **上下文记忆**：AI 会记住之前的对话，可以说"再跑一次"或"用同样的设置"
